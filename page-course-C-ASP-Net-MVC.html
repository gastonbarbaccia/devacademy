C# Asp.Net MVC ( Ajax + Json +Bootstrap + JavaScript)

Aprende Asp.Net MVC y como interactua con las tecnologías mas usadas y conocidas JQuery , Bootstrap entre otras

Aprende a desarrollar aplicaciones web , si vienes del mundo de aplicaciones windows form , es hora que conviertas tu aplicación a una web para que miles de usuarios puedan accederlo desde un browser y así llegue a mas personas en el mundo. Ademas aprenderás como aplicar las librerías mas populares en el mercado JQuery y Boostrap para el desarrollo de aplicaciones , y si no conoces , aquí aprenderás de cero a utilizarlo.

Lo que aprenderás

  • Como funciona el patron de diseño MVC
  • Realizar listados de la base de datos a nuestra aplicación
  • Aprender a transportar y leer información de un JSON
  • Aprender a usar modales para el desarrollo de aplicaciones
  • Como pasar datos de un controlador a una vista
  • Usar Bootstrap para mejorar la interfaz de nuestras aplicaciones
  • Realizar una petición AJAX del cliente al servidor
