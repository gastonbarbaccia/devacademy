Curso MEAN Stack - Crea tus propias Aplicaciones

Lo que aprenderás

  • A utilizar el MEAN Stack a nivel profesional

  • Express

  • NodeJS

  • Autenticar y Autorizar

  • MongoDB

  • Angular

  • JSON Web Token

  • A crear tu propia aplicación full stack

Descripcion del curso

Aprende a crear Aplicaciones Web con Angular, NodeJS, Express y MongoDB. Estas 4 tecnologías juntas forman en MEAN Stack y son una poderosa herramienta hoy en día. Todos los conceptos de este curso se empiezan desde cero hasta alcanzar un conocimiento avanzado de los mismos.

Como puedes observar en el temario de este curso comenzarás aprendiendo Angular, luego pasarás a MongoDB y finalizarás con NodeJS y Express, todos los conceptos se ven por separado para tener amplio conocimiento de cada uno de ellos y por último finalizaremos con una aplicación en la que integraremos el MEAN Stack, con lo que ganarás confianza en crear tus propias aplicaciones.
A lo largo de este curso aprenderás lo siguiente:
1.Angular -> Actualmente el framework para JavaScript más utilizado para desarrollar SPA (Single Page Application) lo que permite crear web modernas y escalables.
2.MongoDB -> Hoy en día la base de datos más utilizada NoSQL que dar mucha funcionalidad a nuestras aplicaciones.
3.NodeJS -> En entorno líder para la creación de servidores, es a la vez ligero y escalable.
4.Express -> Nos permite crear API con node de una manera muy sencilla y además de una manera sólida.
5.Angular Material y así diseñar aplicaciones siguiendo los patrones de estilo de Google.
6.Usar NodeJS y Express de manera eficiente y crear tu servicio REST
7.Conectar tu Servidor con MongoDB y realizar operaciones CRUD
8.Implementar autenticación y autorización a través de JWT.
9.Conectar el FrontEnd (Angular) con tu Backend (NodeJS y Express) y este a su vez con la base de datos (MongoDB)
Hoy en día ser un desarrollador full-stack esta muy bien valorado por parte de las empresas puesto que entienden que conoces la parte del front y la del back y por lo tanto siempre trabajas pensando en optimizar ambos mundos y no solo la parte en la que te toca.
El curso es muy completo y al tratarse en la primera parte todas las tecnológicas por separado, sabrás en cada momento que parte estas tocando y como utilizarla de manera eficiente. Y para terminar ponemos todo el conocimiento encima de la mesa para desarrollar una aplicación real, en la que un frontent diseñado en angular se conecta a un backend en NodeJS + Express y este a la vez maneja los datos en MongoDB.
